## Critical - 즉시 확인 필요

### ✅ 1. 네비게이션 구조

**확인 사항**:

- [ ] 사이드바에 모든 주요 메뉴가 표시되는가?
  - Items (Dashboard)
  - Clients
  - Connected Clients
  - Calendar
  - Sales
  - **Instruments** ⭐ NEW
  - **Client Analytics** ⭐ NEW
- [ ] 각 메뉴 클릭 시 해당 페이지로 정상 이동하는가?
- [ ] 현재 페이지가 사이드바에서 활성화(highlight)되어 표시되는가?
- [ ] 하위 경로(예: `/clients/analytics`)에서도 부모 메뉴가 활성화되는가?
- [ ] 사이드바 접기/펼치기가 정상 작동하는가?
- [ ] 모바일에서 사이드바 오버레이가 정상 작동하는가?

**테스트 시나리오**:

1. 각 메뉴를 클릭하고 URL이 올바르게 변경되는지 확인
2. `/clients/analytics`로 직접 이동했을 때 "Clients" 메뉴가 활성화되는지 확인
3. 브라우저 뒤로가기/앞으로가기 버튼이 정상 작동하는지 확인

### ✅ 2. 원클릭 판매 기능 ⭐ NEW

**확인 사항**:

- [ ] Dashboard에서 각 악기 행의 액션 메뉴(⋮)에 "Sell" 버튼이 표시되는가?
- [ ] "Sold" 상태가 아닌 악기에만 "Sell" 버튼이 표시되는가?
- [ ] "Sell" 버튼 클릭 시 판매 폼이 열리는가?
- [ ] 판매 폼에 다음 정보가 자동으로 채워지는가?
  - [ ] 악기 정보 (선택한 악기)
  - [ ] 고객 정보 (연결된 고객이 있는 경우)
  - [ ] 판매 가격 (악기의 현재 가격)
  - [ ] 판매 날짜 (오늘 날짜)
- [ ] 판매 폼에서 고객과 가격을 수정할 수 있는가?
- [ ] 판매 저장 후 다음이 정상 작동하는가?
  - [ ] 판매 기록이 Sales History에 생성되는가?
  - [ ] 악기 상태가 자동으로 "Sold"로 변경되는가?
  - [ ] 성공 메시지가 표시되는가?
- [ ] 판매 폼 취소 시 모달이 정상적으로 닫히는가?

**테스트 시나리오**:

1. **케이스 1**: 고객과 연결된 악기 판매
   - 고객과 연결된 악기 선택 → "Sell" 클릭
   - 고객 정보가 자동으로 채워지는지 확인
   - 저장 후 판매 기록과 상태 변경 확인
2. **케이스 2**: 고객과 연결되지 않은 악기 판매
   - 고객과 연결되지 않은 악기 선택 → "Sell" 클릭
   - 고객 선택 드롭다운이 비어있는지 확인
   - 고객을 선택하고 저장
3. **케이스 3**: 가격 수정
   - 악기 가격이 1000인 경우 → "Sell" 클릭
   - 판매 가격을 1200으로 수정
   - 저장 후 Sales History에서 1200으로 기록되었는지 확인

## 🔍 특별히 주의해야 할 사항

### ⚠️ 1. 원클릭 판매 기능

**주의사항**:

- 판매 후 악기 상태가 자동으로 "Sold"로 변경되는지 반드시 확인
- 고객과 연결된 악기의 경우 고객 정보가 자동으로 채워지는지 확인
- 판매 기록이 Sales History에 즉시 반영되는지 확인
- 동시에 여러 판매를 처리할 때 충돌이 없는지 확인

### ⚠️ 2. 네비게이션 구조

**주의사항**:

- `/instruments`와 `/clients/analytics` 페이지가 사이드바에서 접근 가능한지 확인
- 하위 경로에서도 부모 메뉴가 활성화되는지 확인
- 브라우저 뒤로가기/앞으로가기가 정상 작동하는지 확인

### ⚠️ 3. Suspense 경계

**주의사항**:

- `/instruments`, `/calendar`, `/dashboard`, `/clients` 페이지에서 로딩 스켈레톤이 표시되는지 확인
- 데이터 로드 중 에러가 발생하지 않는지 확인
- 느린 네트워크 환경에서도 정상 작동하는지 확인

### ⚠️ 4. 가상화 (Virtualization)

**주의사항**:

- Instruments 페이지에서 50개 이상의 악기가 있을 때 가상화가 작동하는지 확인
- 스크롤 시 성능 저하가 없는지 확인
- 가상화된 리스트에서 검색/필터가 정상 작동하는지 확인

### ⚠️ 5. 데이터 일관성

**주의사항**:

- 원클릭 판매 후 악기 상태와 판매 기록이 일관되게 업데이트되는지 확인
- 여러 페이지에서 동일한 데이터가 일관되게 표시되는지 확인
- 실시간 동기화가 정상 작동하는지 확인
