# 사장 입장에서 아쉬운 점 개선안

## 🔴 1순위: 즉시 필요한 기능 (비즈니스 핵심)

### 1. 판매 기록 관리 (Sales History) - 0% 완성

**현재 상태**: 데이터베이스 테이블만 존재, UI/기능 없음

**문제점**:

- ❌ 악기를 판매해도 기록이 남지 않음
- ❌ 판매 통계를 볼 수 없음 (월별/연도별 매출, 인기 제품 등)
- ❌ 고객별 구매 이력을 관리할 수 없음
- ❌ 재고 현황과 실제 판매가 연결되지 않음

**영향**:

- **재무 관리 불가**: 월별 매출 파악 불가
- **재고 관리 어려움**: 어떤 제품이 잘 팔리는지 모름
- **고객 관리 불가**: VIP 고객 식별 불가
- **비즈니스 결정 근거 없음**: 데이터 기반 의사결정 불가

**개선안**:

```typescript
// 예시: 판매 기록 추가 기능
1. Dashboard에서 악기 상태를 'Sold'로 변경할 때 판매 기록 자동 생성
2. Sales 페이지에서:
   - 판매 기록 목록 (날짜, 고객, 악기, 가격)
   - 월별/연도별 매출 차트
   - 인기 제품 TOP 10
   - 고객별 구매 금액 순위
   - 판매 리포트 생성 (PDF/CSV)
```

**우선순위**: ⭐⭐⭐⭐⭐ (즉시 필요)

---

### 2. 이미지 업로드 기능 - 10% (UI만 존재)

**현재 상태**: 이미지 입력 필드 UI만 있고 실제 업로드 로직 없음

**문제점**:

- ❌ 악기 사진을 등록할 수 없음
- ❌ 고객에게 제품 이미지를 보여줄 수 없음
- ❌ 온라인 쇼핑몰/카탈로그 연동 불가
- ❌ 제품 검수 기록 시각화 불가

**영향**:

- **고객 만족도 저하**: 제품 사진 없이 판매 어려움
- **온라인 판매 불가**: 이커머스 연동 불가
- **기록 관리 어려움**: 악기 상태 변화 기록 불가
- **마케팅 제약**: 소셜미디어/홈페이지 활용 불가

**개선안**:

```typescript
1. Supabase Storage 통합
2. 이미지 업로드 컴포넌트 구현
3. 이미지 리사이징/압축 (용량 최적화)
4. 이미지 갤러리 뷰
5. 이미지 삭제 기능
```

**우선순위**: ⭐⭐⭐⭐⭐ (즉시 필요)

---

### 3. 알림 시스템 - 0%

**현재 상태**: 캘린더에서 overdue 표시만 있고 실제 알림 없음

**문제점**:

- ❌ 기한이 지난 작업을 놓칠 수 있음
- ❌ 중요한 작업 마감일을 미리 알 수 없음
- ❌ 고객 약속 일정을 깜빡할 수 있음
- ❌ 이메일/SMS 알림 없음

**영향**:

- **고객 신뢰도 저하**: 약속 일정 누락
- **업무 효율 저하**: 긴급 작업 놓침
- **손실 발생**: 기한 초과로 인한 불만/환불 가능성

**개선안**:

```typescript
1. 작업 마감일 D-3, D-1 알림
2. 기한 초과 작업 일일 알림
3. 이메일 알림 (선택적)
4. 대시보드 알림 배지
5. 알림 설정 (언제, 무엇을 알림받을지)
```

**우선순위**: ⭐⭐⭐⭐ (높음)

---

### 4. 데이터 내보내기/가져오기 - 0%

**현재 상태**: CSV/Excel 내보내기 기능 없음

**문제점**:

- ❌ 회계 프로그램과 데이터 연동 불가
- ❌ 백업을 수동으로 할 수 없음
- ❌ 다른 시스템으로 데이터 이전 불가
- ❌ 엑셀에서 분석 작업 불가

**영향**:

- **회계 작업 비효율**: 수동 데이터 입력 필요
- **백업 위험**: 시스템 장애 시 데이터 손실 가능
- **확장성 제약**: 다른 도구와 연동 불가

**개선안**:

```typescript
1. CSV 내보내기 (Instruments, Clients, Sales)
2. CSV 가져오기 (일괄 등록)
3. Excel 형식 지원
4. 정기 자동 백업 (선택적)
```

**우선순위**: ⭐⭐⭐⭐ (높음)

## ⚠️ 2순위: 중요한 기능 (중기 개선)

### 5. 감사 로그 (변경 이력 추적) - 0%

**현재 상태**: 데이터 변경 이력 추적 없음

**문제점**:

- ❌ 누가, 언제, 무엇을 변경했는지 모름
- ❌ 실수로 삭제된 데이터 복구 불가
- ❌ 데이터 무결성 확인 불가
- ❌ 책임 소재 파악 어려움

**영향**:

- **데이터 신뢰성 저하**: 변경 이력 없음
- **복구 불가**: 실수 삭제 시 복구 어려움
- **감사 어려움**: 문제 발생 시 추적 불가

**개선안**:

```typescript
1. 모든 CRUD 작업 로그 기록
2. 변경 전/후 값 저장
3. 변경 이력 조회 기능
4. 데이터 복구 기능
```

**우선순위**: ⭐⭐⭐ (중간)

### 6. 통계/대시보드 - 20% (기본 데이터만 표시)

**현재 상태**: 단순 목록만 표시, 통계 없음

**문제점**:

- ❌ 매출 통계 없음
- ❌ 인기 제품 파악 불가
- ❌ 고객 분석 불가
- ❌ 재고 현황 요약 없음

**영향**:

- **비즈니스 인사이트 부족**: 데이터 기반 의사결정 불가
- **재고 관리 어려움**: 어떤 제품이 잘 팔리는지 모름
- **마케팅 전략 수립 어려움**: 타겟 고객 파악 불가

**개선안**:

```typescript
1. 매출 통계 (일/주/월/년)
2. 재고 현황 요약 (상태별 개수)
3. 인기 제품 차트
4. 고객 분석 (구매 횟수, 총액)
5. 작업 통계 (완료율, 평균 소요 시간)
```

**우선순위**: ⭐⭐⭐ (중간)
